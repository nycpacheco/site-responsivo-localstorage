Sobre

Oque é IFSP

O Instituto Federal de Educação, Ciência e Tecnologia de São Paulo (IFSP) é uma instituição pública federal voltada ao ensino, pesquisa e extensão. O IFSP oferece cursos técnicos, tecnológicos e superiores em diversas áreas do conhecimento, com foco na formação profissional, científica e tecnológica de seus estudantes.

Oque é ADS?

Análise e Desenvolvimento de Sistemas (ADS) é um curso voltado ao desenvolvimento de softwares, sites e sistemas. A formação envolve programação, banco de dados, lógica computacional e tecnologias utilizadas no mercado de TI.

Sobre esse site

Este site foi desenvolvido como uma atividade acadêmica utilizando HTML, CSS e JavaScript puro. Seguindo as instruções propostas em aula, o projeto possui design responsivo, armazenamento de dados com LocalStorage, formulários interativos para cadastro e consulta de usuários, sistema de contato, integração com mapas reais utilizando Leaflet.js e páginas informativas sobre o curso e a instituição. O site também utiliza funções como map, filter e reduce para manipulação e exibição dinâmica dos dados armazenados no navegador.

Durante o desenvolvimento do projeto, o ChatGPT foi utilizado como ferramenta de apoio, auxiliando na estruturação inicial do site e do código. A IA também ajudou na implementação e explicação de funcionalidades em JavaScript, incluindo LocalStorage, integração com Leaflet.js e o sistema de consulta de usuários. Além disso, foi utilizada para identificar erros no código, sugerir melhorias e auxiliar na criação de textos e descrições para tornar o site mais realista.

Projetos em destaque

Jogo do gato (site)

Site responsivo desenvolvido com HTML, CSS e JavaScript. O objetivo do jogo é descobrir qual número de 1 a 100 o gato escolheu utilizando apenas 6 tentativas. (OBS: site possui som.)

HTML • CSS • JavaScript

Radar espacial em C

Sistema de radar espacial desenvolvido em Linguagem C. O jogador deve escolher uma posição de pouso segura dentro de um campo de asteroides representado por uma matriz 5x5. O radar escaneia as regiões adjacentes e analisa a densidade de asteroides para determinar se o pouso é seguro ou perigoso.

C • Matrizes • Funções

Seleção de candidatos por Idade

Sistema desenvolvido em Portugol para análise de candidatos em vagas de emprego. O programa recebe o nome e idade dos participantes e determina automaticamente quais vagas cada pessoa pode ocupar com base em critérios pré-definidos.

Portugol • Algoritmo • Logica
